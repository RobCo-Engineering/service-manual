# Espruino API Reference

Outside of the [`Pip`](./pip) library, many of the available methods built into
the Pip-Boy are part of the standard set of functions that ship with Espruino.
As such it's worth referencing the Espruino Reference directly to save
replicating information:

- [https://www.espruino.com/Reference](https://www.espruino.com/Reference)
